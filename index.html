Protocolo LBH

Lenguaje Binario HormigasAIS — Especificación técnica v2.0

GET api.hormigasais.com/verify

Verifica que el nodo soberano A16 está activo y retorna la firma maestra del protocolo.

{ "status": "VALIDADO", "signature": "CLHQ-MASTER-KEY", "protocol": "Lenguaje-Binario-HormigasAIS", "origin": "Nodo-Soberano-A16" }
POST api.hormigasais.com/seal

Registra un sello criptográfico para un activo digital. Retorna una firma única CLHQ verificable.

// Request { "owner": "Nombre-Propietario", "asset": "descripcion-del-activo", "hash": "sha256-del-archivo" } // Response { "status": "SELLADO", "sello": { "signature": "CLHQ-A8F3K2M1", "protocol": "Lenguaje-Binario-HormigasAIS", "timestamp": "2026-05-08T..." } }
GET api.hormigasais.com/seal/{firma}

Consulta y verifica la autenticidad de un sello emitido usando su firma CLHQ.

// Response si existe { "status": "VERIFICADO", "sello": { ... } } // Response si no existe { "error": "Sello no encontrado" }
POST api.hormigasais.com/push/{nodo}

Envía una feromona de aprendizaje desde un nodo de la colonia.

{ "type": "learning", "value": 0.85, "trust": 0.9 }
